
## Project Description
This project is created as part of Assignment 1 for the IT3040 â€“ ITPM module.  
The objective of this assignment is to evaluate the accuracy and usability of a real-world web application that converts **Singlish input into Sinhala output**.

The system under test:
https://www.swifttranslator.com/

Automated functional and UI-related test scenarios have been implemented using **Playwright**.


## Technologies Used
- Node.js
- Playwright
- TypeScript
- Git & GitHub

## Prerequisites

- Node.js (version 18 or above recommended)
- npm (comes with Node.js)


## Installation Steps
1. Clone the repository:
git clone https://github.com/DanushiKaveesha/ITPM_Assignment_IT23709898.git

2. Navigate to the project folder:

cd ITPM_Assignment_IT23709898

3. Install dependencies:

npm install

4. Install Playwright browsers:

npx playwright install



## Running the Tests


npx playwright show-report


## Test Coverage

The automated test cases cover:

* Positive and negative functional scenarios
* Different sentence structures (simple, compound, complex)
* Interrogative and imperative forms
* Input length variations (short, medium, long)
* Mixed Singlish and English inputs
* Slang and informal language
* UI-related behavior (real-time Sinhala output updates)
